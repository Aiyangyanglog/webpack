## 1.什么是webpck
* webpack是模块打包机: 分析你的项目结构，找到JavaScript模块及其他的一些浏览器不能直接运行的拓展语言（Scss,TypeScript等），并将其打包为合适的格式以供浏览器使用
* 可以做的事情:代码转换,文件优化，代码分割，模块合并，自动刷新，代码校验，自动发布
